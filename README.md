# K295 Grant Dashboard — Virridy

Live: [https://virridy-nsf-dashboard.pages.dev](https://virridy-nsf-dashboard.pages.dev)

Static dashboard for tracking NSF grant K295 progress (Contract 49100425C0009). Displays historical milestone data from monthly reports across 9 objectives.

## Views

- **Overview** — Summary card + 3x3 objective grid with sparklines
- **Objective Detail** — Per-milestone progress charts (click any objective card)
- **Timeline** — Horizontal timeline with collapsible snapshot table

## Stack

Static HTML/CSS/JS, Chart.js (CDN), deployed on Cloudflare Pages.
